# معرض أعمال أحمد قاعود - Ahmad Qaoud Portfolio

هذا المشروع هو معرض أعمال احترافي (Premium Class A) يعرض خدمات ومنتجات أحمد قاعود في مجالات التحول الرقمي والذكاء الاصطناعي.

## هيكلية المشروع الجديدة

تمت إعادة تنظيم المشروع ليتبع أفضل الممارسات المهنية:

- **[Frontend (React/Vite)](file:///e:/ALL%20PROGECT%20AI/your-personal-prelude-main/your-personal-prelude-main/frontend)**: الكود المصدري للواجهة الأمامية.
- **[Backend (Supabase)](file:///e:/ALL%20PROGECT%20AI/your-personal-prelude-main/your-personal-prelude-main/backend)**: إعدادات قاعدة البيانات والمنطق الخلفي.
- **[Documentation](file:///e:/ALL%20PROGECT%20AI/your-personal-prelude-main/your-personal-prelude-main/docs)**: التوثيق الكامل للمشروع وملفات التعريف.

---

## كيف تبدأ؟

1. توجه إلى مجلد الواجهة الأمامية: `cd frontend`
2. ثبت الاعتمادات: `npm install`
3. شغل خادم التطوير: `npm run dev`

لمزيد من التفاصيل، يرجى مراجعة [مركز التوثيق](file:///e:/ALL%20PROGECT%20AI/your-personal-prelude-main/your-personal-prelude-main/docs/README.md).
